LATITUDE VS. WEATHER

A Study Looking At The Relationship Between Latitude And Several Weather Phenomenon

An analysis was done looking at weather data for 500 + random cities to see if there were any discernable weather trends that varied with latitudes.

To accomplish this analysis, we generated a list of over 500 randomly selected cities and we pulled weather data for each city using the OpenWeatherMap API to assemble the dataset.

Once the dataset was assembled, we then utilized Matplotlib to make various plots of our weather attributes vs latitudes. For further analysis and to check for repeatability, we split our data into Nothern hemisphere data and southern hemisphere data to see if any observed trends were applicable in the data for each hemisphere.

The results of this analysis were compiled in this site. Along with the source data and our observed trends.
